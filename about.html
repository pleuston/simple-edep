About simple-edep

simple-edep is a lightweight, browser-based editor for Roman and Latin inscriptions. It encodes inscriptions as EpiDoc/TEI XML and stores them in a public GitHub repository — there is no server and no database. The XML is built entirely in your browser and committed through the GitHub Contents API with your own access token.

Scope

The corpus collects Roman/Latin inscriptions — funerary, votive, honorific, building and milestone texts among others — each encoded with object and support description, layout and lettering, dating, find spot, edition, apparatus, translation, commentary and bibliography, following the EpiDoc Guidelines. Records cross-reference the major databases (EDH, EDCS, EDR, Trismegistos, PHI, CIL) and use authority vocabularies from EAGLE, the Getty AAT and Pleiades.

How it is built

The editing model follows the EDEP project at Heidelberg (edep.adw.uni-heidelberg.de) and the open-source eeditiones/edep framework, reimagined as a static site so it can run from GitHub Pages alone. The live Leiden+ → EpiDoc transcription editor is the jinn-codemirror component, the same Leiden engine used by EDEP. The application structure is modelled on the epiwen token-based editing framework.

Sections
