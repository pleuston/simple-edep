simple-edep EpiDoc editor for Roman epigraphy
Docs

A browser-based editor that turns Leiden+ transcriptions into EpiDoc/TEI XML and commits them straight to GitHub with your own token — no server, no database. Built on the jinn-codemirror Leiden engine, modelled on the epiwen framework.

Editor →
Encode an inscription with the live Leiden+ ↔ EpiDoc editor and save to GitHub.
Catalogue →
Browse, search and re-edit the inscriptions already in the corpus.
Map →
See find spots on a map; click a marker to read the inscription.
Client-side only · EpiDoc/TEI · records committed to pleuston/simple-edep. Leiden conventions follow the EpiDoc Guidelines (epidoc.stoa.org).
