simpleEDEP

Edition tools for Roman digital epigraphy

A browser-based editor for Roman/Latin inscriptions: a structured metadata form and a live Leiden+ → EpiDoc editor, with records stored in GitHub. It is a simple rewrite of the EDEP edition tool (eeditiones/edep) for serverless deployment — no database, no application server.
