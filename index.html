simple-edep Edition tools for Roman digital epigraphy
Inscriptions Geodata People About Instructions

A browser-based editor that turns Leiden+ transcriptions into EpiDoc/TEI XML and commits them straight to GitHub with your own token — no server, no database. Modelled on the EDEP framework (eeditiones/edep), with the same jinn-codemirror Leiden engine.

Editor →
Encode an inscription with the live Leiden+ ↔ EpiDoc editor and save to GitHub.
Inscriptions →
Browse, search and re-edit the inscriptions in the corpus.
Geodata →
Find spots plotted on a map; click a marker to read the text.
People →
Register of persons named in the inscriptions.
Client-side only · EpiDoc/TEI · records committed to pleuston/simple-edep. See About and Instructions.
